# 學生座位排程系統功能說明

## 功能概述
學生座位排程系統現在支援可自行設計的列印字體大小和班級名稱功能，讓使用者可以根據需要自由輸入列印時學生姓名的字體大小，以及自由設定班級名稱（如課後班、正規班、補習班等）。同時，講台在列印時會清楚顯示，提升列印品質。

## 新增功能

### 1. 可自行設計字體大小控制介面
- 在設定區域新增了「列印設定」區塊
- 提供數字輸入框，可自由輸入 8-48px 之間的字體大小
- 支援精確的數值控制，不再受限於固定步長
- 包含說明文字，清楚說明功能用途
- 美觀的數字輸入框設計，支援鍵盤輸入

### 2. 班級名稱設定介面
- 在設定區域新增了「班級設定」區塊
- 提供文字輸入框，可自由輸入班級名稱
- 支援最多 30 個字元的班級名稱
- 預設班級名稱：課後班
- 班級名稱會即時顯示在頁面標題

### 3. 講台列印改善
- 深色背景配合白色邊框，確保列印時清楚可見
- 增加邊框粗度和圓角，提升視覺效果
- 金色圖示和白色文字，形成強烈對比
- 漸層邊框效果，增加美觀性
- 更大的字體和間距，提升可讀性
- 陰影效果，增加立體感
- 專門的列印樣式，確保黑白列印時仍清楚可見

### 4. 字體大小設定
- 預設字體大小：16px
- 可調整範圍：8px - 48px（大幅擴展）
- 支援自由輸入，不再受限於固定選項
- 自動範圍驗證，超出範圍會自動調整
- 設定會自動儲存到瀏覽器本地儲存

### 5. 應用範圍
字體大小、班級名稱和講台樣式設定會應用到以下功能：
- **列印座位表**：學生姓名字體大小、班級名稱標題、清楚顯示的講台
- **匯出 Word**：學生姓名字體大小、班級名稱標題、清楚顯示的講台
- **頁面顯示**：頁面標題顯示當前班級名稱

## 技術實現

### HTML 修改
- 在 `index.html` 的設定區域新增字體大小控制和班級名稱設定
- 使用 `number` 輸入類型提供精確的字體大小控制
- 使用 `text` 輸入類型提供班級名稱輸入
- 添加適當的標籤和說明文字
- 在頁面標題新增班級名稱顯示區域

### CSS 樣式
- 新增 `.font-size-control` 樣式類別
- 新增 `.class-display` 樣式類別
- 改善講台列印樣式，提升可見性
- 自定義數字輸入框外觀，移除預設的上下箭頭
- 響應式設計，適配不同螢幕尺寸
- 美觀的班級名稱顯示樣式

### JavaScript 功能
- 新增 `printFontSize` 屬性儲存字體大小設定
- 新增 `className` 屬性儲存班級名稱設定
- 實現 `updateFontSize()` 方法處理字體大小變更，包含範圍驗證
- 實現 `updateClassName()` 方法處理班級名稱變更
- 整合到本地儲存系統，自動保存和載入設定
- 修改列印和匯出功能，使用設定的字體大小和班級名稱
- 即時更新頁面標題顯示
- 改善講台在列印和匯出時的顯示效果

## 使用方法

### 1. 調整字體大小
- 在設定區域找到「列印設定」
- 在數字輸入框中輸入字體大小（8-48px）
- 支援鍵盤直接輸入或使用上下箭頭微調
- 超出範圍會自動調整到有效範圍內

### 2. 設定班級名稱
- 在設定區域找到「班級設定」
- 在輸入框中輸入班級名稱（如：課後班、正規班、補習班等）
- 班級名稱會即時顯示在頁面標題
- 支援最多 30 個字元

### 3. 講台列印效果
- 講台會以深色背景和白色邊框顯示
- 列印時會自動調整為適合列印的樣式
- 黑白列印時仍會清楚顯示
- 講台位置會自動避免分頁

### 4. 自動儲存
- 調整後設定會自動儲存
- 重新載入頁面後設定會保持

### 5. 應用效果
- 列印座位表時會使用設定的字體大小、班級名稱和清楚顯示的講台
- 匯出 Word 文件時也會使用設定的字體大小、班級名稱和清楚顯示的講台
- 頁面標題會即時顯示當前設定的班級名稱
